# 攻略チャート 1 トーティスの村〜ローンヴァレイ

## 1. トーティスの村

### 狩りのため南の森へ

オープニング後、トーティスの村にフェーズが移ります。イベント中にはクレスの母であるマリアから「アップルグミ」をもらいます。その後、狩りに行くためにトーティスの村からフィールドに出て、南にある森へ向かいます。

<Tips title="アミィの贈り物">

    }
</Tips>

<Tips title="おかみの頼みごと">

## 2. 南の森

### 森のイノシシを追いかける

イノシシを追って森の奥まで行くとボス「ボア、ボアチャイルド ×3」と戦闘となります。ボアと距離をおいてからのダッシュ攻撃を行うことで苦戦することなく、勝利することができるはずです。

勝利後、トーティスの村へ戻るとイベントが発生する。イベント後はクレスの叔父に助けを求めるために、北にある山道を通過してユーグリッドの都に向かう。

### ボス：「ボア、ボアチャイルド ×3」

| 項目           | データ       |
| -------------- | ------------ |
| HP             | 175          |
| 弱点           | 火           |
| 落とすアイテム | ポーク(100%) |

<Tips title="最序盤の回復方法">
    {
        <>
            これまでの過程でHPが赤く表示されたり味方キャラクターが戦闘不能の場合は無理をせずにトーティスの村に戻って回復しましょう。チェスターの家1階のぐつぐつしている鍋を調べるとマーボーカレーを食べることができHPとTPを回復できます。道場2階にあるクレスの部屋のベットでもHPとTPの回復が可能。
        </>
    }
</Tips>
