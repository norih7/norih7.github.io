# 攻略チャート 1 トーティスの村〜ローンヴァレイ

## 1. トーティスの村

### 狩りのため南の森へ

オープニング後、トーティスの村にフェーズが移ります。イベント中にはクレスの母であるマリアから「アップルグミ」をもらいます。その後、狩りに行くためにトーティスの村からフィールドに出て、南にある森へ向かいます。

## 2. 南の森

### 森のイノシシを追いかける

イノシシを追って森の奥まで行くとボス「ボア、ボアチャイルド ×3」と戦闘となります。ボアと距離をおいてからのダッシュ攻撃を行うことで苦戦することなく、勝利することができるはずです。

勝利後、トーティスの村へ戻るとイベントが発生する。イベント後はクレスの叔父に助けを求めるために、北にある山道を通過してユーグリッドの都に向かう。

### ボス：「ボア、ボアチャイルド ×3」

| 項目           | データ       |
| -------------- | ------------ |
| HP             | 175          |
| 弱点           | 火           |
| 落とすアイテム | ポーク(100%) |

## 3. 山道〜ユーグリッドの都

### 山道を通過してユーグリッドの都へ

ユーグリッドの都に行くにはトーティスの北にある山道を通っていくことになります。山道には石像が置いてありこのままだと通れませんが、決定ボタン（○ボタン/Aボタン)を押しながら方向キーで操作すると石造を移動することができ、通れるようになります。ちなみに石像を動かさなくても左の道を選んでも道を抜け可能ですがアイテムを入手できません。

### オルソンの家で休む
ユーグリッドの都についたらメイン通路をまっすぐ進んで左にあるオルソンの家に行きオルソンに話しかけ、「休む」を選択します。ただし、この選択肢を選ぶとユーグリッドには戻ってこられなくなるのでアイテムなどの準備をしておきましょう。特にロールキャベルのレシピはここでしか入手できません。
